# 学習マップ（本書の位置づけ：暫定）

信号処理は「時間領域↔周波数領域」「設計↔評価」を往復して理解が深まる。
本書は **決定論（＋簡単な雑音）** と **時間↔周波数↔設計** を，ツール無しで貫く。

## 本書で扱う（10章）
- 信号・サンプリング・量子化
- LTIと畳み込み（Toeplitz行列）
- DTFT/DFT/FFT（DFT行列）
- 窓とスペクトル推定（平均化PSD）
- STFT（スペクトログラム）
- Z変換と安定性（IIRの入口）
- FIR設計（窓法）と評価
